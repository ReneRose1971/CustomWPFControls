# CustomWPFControls

Ein MVVM-Framework für WPF-Anwendungen mit automatischer DataStore-Synchronisation, wiederverwendbaren Controls und Dependency Injection.

## Überblick

**CustomWPFControls** ist eine Library für WPF-Entwicklung mit .NET 8, die ViewModels, Collection-Synchronisation, Editor-Controls und Services bereitstellt.

## Projekte in dieser Solution

### [CustomWPFControls](CustomWPFControls/README.md)
Haupt-Library mit ViewModels, Controls, Commands und Services.

### [TestHelper.CustomWPFControls](TestHelper.CustomWPFControls/README.md)
Test-Utilities und Fixtures für Unit- und Integrationstests.

### CustomWPFControls.Tests
Test-Suite für die Library.
